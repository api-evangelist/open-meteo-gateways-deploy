Open-Meteo × agentgateway

agentgateway turns the two free, keyless Open-Meteo REST APIs into one federated MCP server. Each OpenAPI operation becomes an MCP tool; the two targets are multiplexed into a single tool list. One of three gateways over the same upstreams — see also Tyk and KrakenD.

This is MCP, not REST — the consumer is an AI agent. There is no REST surface here. Point an MCP client at the endpoint below (a plain browser GET / only reaches this page).

MCP endpointPOST https://weather-agentgateway.apievangelist.com/mcp (Streamable HTTP)
Toolsweather_getWeatherForecast, airquality_getAirQuality

Try it with MCP Inspector:

npx @modelcontextprotocol/inspector
# then connect to: https://weather-agentgateway.apievangelist.com/mcp
